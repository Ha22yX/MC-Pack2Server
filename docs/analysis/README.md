# Analysis Artifacts

This folder stores sample-driven research for Pack2Serve.

## Reports

- `reports/BattleArmory_TACZ_mrpack_analysis.md` - Single-pack analysis for a Forge Modrinth pack.
- `reports/mrpack_common_patterns_analysis.md` - Shared patterns across two Modrinth packs.
- `reports/expanded_modpack_patterns_5_samples.md` - Expanded comparison across Modrinth, CurseForge, Forge, Fabric, and NeoForge samples.

## CSV Inventories

- `manifests/` contains remote dependency listings from Modrinth or CurseForge manifests.
- `overrides/` contains ZIP override-layer inventories.

These files are development references for designing the parser, server builder, and runtime manager.
